- `ARCHITECTURE.md` - Design decisions, components, data flow
- `IMPLEMENTATION.md` - Code changes, technical details
- `TEST_PLAN.md` - Manual test cases, performance benchmarks
- `CHANGELOG.md` - Version history, breaking changes
